Detailed Description

template<bool Contiguous, typename PointType, typename DifferenceType>
class pss::astrotypes::multiarray::DimensionMeta< Contiguous, PointType, DifferenceType >

Provides information about what the bins in a given dimension represene.

Therse are represented as either a starting valuae and an equal offset for each subsequenct bin or as a vector with a value assigned for each bin.

Parameters
Contiguoustrue : the start_point + offset model false: the vector of values model (could be, bit not necessarily is contiguous)
PointTypeThe representational quantity to specify a point in the requred dimesnon (e.g seconds since an epoc for time)
DifferenceTypeThe type representing the difference between adjacent PointTypes
