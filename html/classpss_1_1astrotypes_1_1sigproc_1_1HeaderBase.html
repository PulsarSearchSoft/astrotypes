Detailed Description

template<typename Derived>
class pss::astrotypes::sigproc::HeaderBase< Derived >

Store SigProc header information.

Sigproc has a number of mandatory and optional parameters All stored at the beginning of the sigproc file.

As a consequence the type often returned is a utils::Optional<T>. This class is designed to allow you to test if the value has been set or not. e.g.

if(!header.source_name()) {
// oh oh no source name, better deal with it
...
}
// once tested you can get the underlying variable with the dereference operator *
std::string const& source = *header.source_name();

note also that many of the types returned as types with specific units. If your writting new code its a good idea to keep this type information when you pass this value around, but for compatibility with less type safe codes and you need just the numerical value you can call the value() mehtod

// n.b. this loses type information.
double tsamp_in_secs = header.sample_interval().value();
