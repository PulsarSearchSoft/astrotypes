Detailed Description

template<typename T, typename Alloc = std::allocator<T>>
class pss::astrotypes::FrequencyTime< T, Alloc >

A template class representing values associated with a time and frequecny such as Stokes values or voltages.

Stored as a contiguous channels in time. This can be used in exactly the same calls as the TimeFrequency object. They are designed to be interchangable without having to rewrite any code that uses this interface.
