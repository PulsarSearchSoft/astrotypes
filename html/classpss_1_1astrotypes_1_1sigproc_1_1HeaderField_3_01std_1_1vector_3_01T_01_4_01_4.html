Detailed Description

template<typename T>
class pss::astrotypes::sigproc::HeaderField< std::vector< T > >

specialisation for writing and reading the sigproc vector type

A start of a vector in the header is marked with a start_label (e.g. FREQUENCY_START) A end of a vector in the header is marked with an end_label (e.g. FREQUENCY_END) In between each element is marked with its own item_tag (e.g. fchannel)
